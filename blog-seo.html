SEO definition


SEO est l'abréviation de "Search Engine Optimization", c'est-à-dire optimisation pour les moteurs de recherche. Il s'agit de techniques qui permettent à votre site web d'être mieux classé dans les pages de résultats des moteurs de recherche (SERP).

En étant plus visible pour les personnes vous touchez davantage de personnes susceptibles d'être intéressées par vos produits, vos services. Pour rappel les moteurs de recherche les plus connus sont: Google, Bing, Yahoo, Yelp, Youtube.

Les techniques que nous utilisons pour améliorer le classement des sites web sont en constantes évolution car les algorithmes de recherche changent constamment.
